You should find a repeating sequence inside the substring. I have an example for you: in a string "abababc" - "ab" is a sequence that repeats more than once, so the answer will be "ababab"

Input: String.

Output: String.

Example:

repeatInside('abc') == ''
repeatInside('abcabcabab') == 'abcabc'
